# `@scodx/prettier-config`

> My personal [Prettier](https://prettier.io) config.

Based on [azz's example](https://github.com/azz/prettier-config)

## Usage

**Install**:

```bash
$ yarn add --dev @scodx/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@scodx/prettier-config"
}
```
